Welcome to Formlytic

Formlytic is a UI library kit that allows you to transform standard form elements into advanced, dynamic, and searchable components with ease.

Dynamic Selects

Transform your basic <select> elements into searchable dropdowns with just an attribute.

Prefix and Suffix Support

Easily add prefixes and suffixes to input fields for better context and usability.

Lightweight and Easy to Use

No complex setup. Just include the library and enhance your form elements instantly.

How It Works

Simply add the searchable="true" attribute to your <select> elements, or use attributes for prefix and suffix, and the library will handle the rest!

<select searchable="true">
    <option value="option-1">Option 1</option>
    <option value="option-2">Option 2</option>
</select>
